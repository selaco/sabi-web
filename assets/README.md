# Assets SABI

Carpeta preparada per afegir imatges pròpies del restaurant.

Noms suggerits ja referenciats com a `data-asset` a `index.html`:

- `hero-barra.jpg` - barra íntima amb cuina vista i ombres càlides.
- `ceramica-artesanal.jpg` - detall de ceràmica artesanal o vaixella irregular.
- `fusta-fosca.jpg` - textura de fusta fosca, barra o mobiliari.
- `producte-temporada.jpg` - producte de mercat, peix, verdura o fruita de temporada.

La web actual funciona sense aquestes imatges: `styles.css` genera placeholders elegants amb gradients i textures. Quan hi hagi fotografies finals, pots afegir-les aquí i substituir els fons CSS corresponents per `url("assets/nom-fitxer.jpg")`.
